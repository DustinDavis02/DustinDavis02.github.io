SkyRunner

Sky Runner is a 2D platformer game developed using Unity. In the game, players control a character that runs and jumps across various levels while avoiding obstacles and enemies. The objective of the game is to reach the end of each level while collecting melons and avoiding traps. The player character can run and jump using the arrow keys or AD keys and spacebar on the keyboard.
